# Git 協作練習

## 步驟
1. 複製專案到本機

  ```
  $ git clone https://github.com/104corp/git-practice-101.git
  ```

2. 新增分支

  ```
  $ git branch -b <自己取一個分支名稱>
  ```

3. 修改 `docs/list.md` 內容並存檔
4. commit 檔案變更

  ```
  $ git add .
  $ git commit
  ```

5. 上傳分支

  ```
  $ git push
  ```

6. 在 Github 上發起 Pull Request
